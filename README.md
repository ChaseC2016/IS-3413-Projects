# IS-3413-Projects
Course project portfolio from IS-3413 — exploring core information systems concepts through practical labs in IoT configuration, network simulation, and device management with Cisco Packet Tracer and more.

# Lab 01 – IP Addressing, Binary Conversion & Subnet Analysis
Identified IPv4, IPv6, and MAC addresses from a live Windows host using ipconfig, converted each to binary, and performed bitwise AND operations with standard and inverted subnet masks to derive network and host addresses. Extended the exercise using a Python-generated random Class C IP address.

# Lab 04 – TCP vs. UDP Traffic Analysis with Wireshark
Analyzed a packet capture (PCAP) file in Wireshark to compare the structural and behavioral differences between TCP and UDP at the transport layer. Inspected UDP and TCP header fields, traced the TCP three-way handshake (SYN/SYN-ACK/ACK), and examined TCP congestion control mechanisms including slow start, fast retransmit, and timeout-based recovery.

# Lab 05 – Network Fundamentals & Device Configuration
Completed a series of hands-on Cisco Packet Tracer activities covering structured cabling and rack configuration, wireless device connectivity (Wi-Fi, Bluetooth, mobile hotspot), CLI-based device management via console cable, packet flow analysis using simulation mode, network topology editing, centralized network monitoring through a Cisco Network Controller, and wireless fault diagnosis using WPA2-PSK authentication.

# Lab 07 – Smart Home Network Simulation & IoT Device Configuration
Built and managed simulated smart home environments in Cisco Packet Tracer, connecting and controlling IoT devices including sensors, speakers, doors, and cameras through a home gateway and registration server. Created a custom IoT device from scratch, modified device scripts in JavaScript to implement motion-detection behavior, and explored environmental controls that respond to real-time changes in temperature, water level, and smoke.
